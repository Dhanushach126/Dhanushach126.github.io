GitHub Pages Hosting Information

Note: Your website is hosted on GitHub Pages, which doesn't support PHP. This means the contact form can't use the PHP backend.

How to use the contact form:

Since GitHub Pages doesn't support PHP processing, we've configured your contact form to use Formspree instead. Your form will still work perfectly!

Formspree is already set up:

  • Your form automatically submits to Formspree at https://formspree.io/f/mnnvpabo
  • Messages will be sent directly to your email
  • No server configuration required

If you prefer to use your PHP contact form, you'll need to move to a hosting provider that supports PHP (see options below).
